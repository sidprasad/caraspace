# Releasing

Maintainer release process is in
[RELEASING.md](https://github.com/sidprasad/caraspace/blob/main/RELEASING.md).
It covers the OIDC-based trusted-publishing flow, the order in which the
two crates (`caraspace_export_macros` and `caraspace`) must publish, and
the troubleshooting steps when one of the two publishes lands and the
other doesn't.

If you're not a maintainer of caraspace, this page is unlikely to be
useful — feel free to skip to [Contributing](./contributing.md).
